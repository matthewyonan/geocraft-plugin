GeoCraft connects your WordPress site to the GeoCraft AI GEO content platform for automated publishing and analytics.

== Description ==

GeoCraft connects a WordPress site to the GeoCraft AI GEO content platform, enabling automated content publishing, analytics collection, and SEO meta injection.

Features:

* One-click and bulk content publishing via REST API
* SEO meta injection for Yoast, Rank Math, and AIOSEO
* Analytics collection (pageviews, time-on-page, bounce rate)
* Category and tag mapping between GeoCraft and WordPress
* Featured image sync
* Scheduled publishing support
* AES-256-CBC encrypted API key storage
* Dashboard analytics widget

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/geocraft` directory, or install through the WordPress plugin screen.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Go to Settings > GeoCraft and enter your API key from your GeoCraft dashboard.
4. Configure default post status, author, and category preferences.
