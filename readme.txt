Publish AI-optimized content from GeoCraft directly to your WordPress site. One-click publishing, SEO metadata, and performance analytics.

== Description ==

**GeoCraft connects your WordPress site to the GeoCraft AI content platform for effortless publishing and analytics.**

GeoCraft is an AI-powered content creation platform that produces articles optimized for Generative Engine Optimization (GEO) — designed to rank in traditional search and get cited by AI answer engines like ChatGPT, Perplexity, and Google AI Overviews.

This plugin gives you a direct connection between GeoCraft and your WordPress site, so content goes from creation to publication without manual steps.

= One-Click Publishing =

Push articles from GeoCraft to your WordPress site as drafts or published posts. All formatting, headings, and structure are preserved.

= Automatic SEO Metadata =

Meta titles, descriptions, and schema markup are set automatically. Compatible with Yoast SEO, RankMath, and All in One SEO.

= Featured Image Sync =

Featured images are uploaded and attached to posts automatically.

= Category & Tag Mapping =

Map your GeoCraft content categories to your WordPress taxonomies so articles are organized from the moment they land.

= Scheduled Publishing =

Queue posts for future publication using WordPress's native scheduling.

= Performance Analytics =

Pageviews, time-on-page, and engagement metrics flow back to your GeoCraft dashboard so you can see what's working.

= Bulk Publishing =

Publish multiple articles in a single batch.

== Installation ==

1. Install and activate the plugin from the WordPress Plugin Directory, or upload the ZIP via Plugins > Add New > Upload Plugin.
2. Go to **Settings > GeoCraft** in your WordPress admin.
3. Enter your API key (found in your GeoCraft dashboard under **Integrations**).
4. Configure your default post status (draft or published) and category mappings.
5. Start publishing!

== Frequently Asked Questions ==

= Do I need a GeoCraft account? =

Yes. The plugin requires an active GeoCraft subscription (any paid plan) to connect to the platform.

= Which SEO plugins are supported? =

The plugin automatically detects and works with Yoast SEO, RankMath, and All in One SEO. If no SEO plugin is installed, metadata is stored in standard post meta.

= Is the plugin free? =

The plugin is free to install. An active GeoCraft subscription is required to use it.
