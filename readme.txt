GeoCraft plugin for AI GEO content publishing and analytics integration.

== Description ==

GeoCraft Plugin connects a WordPress site to GeoCraft's AI GEO content platform.

Features:

* Plugin lifecycle hooks for activation/deactivation
* Core settings registration for API configuration
* CI/CD baseline for quality checks and release packaging

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/geocraft-plugin` directory, or install through WordPress plugin screen.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Configure plugin settings when admin pages are added.
